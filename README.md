## Commit Convention
|Convention|Mean|
|:---:|:---|
|FEAT|새로운 기능 구현|
|FIX|버그, 오류 수정|
|ADD|파일 생성, 라이브러리 추가 등|
|CHORE|코드 수정|
|DEL|코드 삭제|
|DOCS|문서 수정|
|TEST|테스트 코드 작성|
|STYLE|코드 포맷팅, 코드 변경이 없는 경우|
|RENAME|파일, 폴더명 수정 및 이동|
|REMOVE|파일 삭제|
|REFACTOR|리팩토링|
|TODO|Todo 작성|
|COMMENT|주석 작성|
